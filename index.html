CMLL

A Machine Learning Library for C++

A Machine Learning library built using just standard C++ from scratch. CMLL uses Standard Template Library with C++17 Standards. The computational requirements are also written from roots. File Handling and manipulation tools are also included making it a fully independent library

COVID-19 Detection

Using deep learning to detect COVID-19 from X-ray Images.

Prediction of Covid-19 positive from X-rays. The images in the Dataset used are Denoised with Convolutional Variational AutoEncoder (hyperparameterized for Covid-19 Dataset) and a Convolutional Neural Network is used for classification with softmax entropy loss.
